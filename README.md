## Participantes

## Descripción general
Funciona como un repositorio de datos informativo en el cual cada usuario puede puede publicar sus propios post y ver los de los demás. 
Se podrá filtrar los post por categoria/tópico.
Se podrá consultar la veracidad de un post mediante IA (implementada dentro de la app).

El proyecto promueve la verificación responsable de la información y el pensamiento crítico, ofreciendo un entorno educativo, participativo y seguro.
