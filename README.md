## Descripción general

El objetivo principal es funcionar como una red social de curiosidades donde todo lo que se publica pasa por el filtro de una inteligencia artificial, fomentando así la verificación de la información e impidiendo la divulgacion de fake news.
Ofrece un juego de preguntas y respuestas tipo quiz, con dos modos de juego (CLASICO Y SUPERVIVENCIA), el cual reutiliza los datos ingresados por los usuarios y genera el contenido verificando y adaptando la informacion con IA.
Cuenta con un sistema de puntos donde los usuarios sumaran y restaran puntos segun la veracidad de los datos publicados o jugando al quiz, con los cuales podran desbloquear trofeos y competir en el ranking.

cd backend
npm i
cd ..

cd frontend
npm i
cd ..

(Tener abierto Docker Desktop)

docker-compose up --build
